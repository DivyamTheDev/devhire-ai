# DevHire AI 🚀

AI-powered job hiring platform with:

- Role-based authentication (Recruiter / Developer)
- Job posting and application system
- Resume upload & viewing
- MongoDB Atlas cloud integration
- AI-powered resume scoring

## Tech Stack
- Node.js
- Express
- MongoDB Atlas
- EJS
- Multer
- AI API (Groq/OpenAI)

## Features
- Recruiter dashboard
- Developer job browsing
- Resume evaluation
- Skill matching score

## Future Improvements
- Job search filter
- Resume parsing
- Dashboard analytics
